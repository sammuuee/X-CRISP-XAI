# Formulaire d'analyse genomique Vue

Application Vue 3 + Vite pour le parcours X-CRISP-XAI.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`

## Routes

- `/` : portail de connexion
- `/subscribe` : inscription et verification SIRET
- `/workspace` : espace de travail
- `/submit` : formulaire simple
- `/results` : resultats XAI
